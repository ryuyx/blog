# Blog

A personal blog built with [Astro](https://astro.build/).

## Commands

| Command          | Action                                           |
| :--------------- | :----------------------------------------------- |
| `pnpm install`   | Installs dependencies                            |
| `pnpm dev`       | Starts local dev server at `localhost:4321`       |
| `pnpm build`     | Build the site                                   |
| `pnpm preview`   | Preview build locally                            |
